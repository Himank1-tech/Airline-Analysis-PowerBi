## Overview

This project is a **comprehensive Airline Analysis Dashboard** created by **Himank Batra** using **Microsoft PowerBI**. It delivers interactive visualizations and actionable insights into airline operations, covering flight performance, passenger statistics, delays, popular routes, and operational KPIs to aid stakeholders in data-driven decision-making.

## 🚀 Features

| Feature | Description |
|---------|-------------|
| **Interactive Dashboards** | Drill-down, slicers, filters for dynamic data exploration |
| **Key Metrics (KPIs)** | On-time performance, load factor, cancellation rates |
| **Advanced Visuals** | Maps, charts, gauges, tables |
| **AI-Powered Insights** | Trends, forecasts, key influencers |
| **Mobile-Optimized** | Responsive design for all devices |

## 🛠️ Tech Stack

- **PowerBI Desktop**
- **DAX** for calculations
- **Power Query** for ETL
- Data: CSV/Excel/SQL

## Quick Start

### Prerequisites
- [PowerBI Desktop](https://powerbi.microsoft.com/desktop/) (free)

### Installation
```bash
git clone https://github.com/Himank1-tech/Airline-Analysis-PowerBi.git
cd Airline-Analysis-PowerBi
```

1. Open `Airline analysis p1.pbix` in PowerBI Desktop
2. Refresh data connections
3. Interact with the dashboard!

Publish: File → Publish to PowerBI Service.

## 💡 Usage
- Filter with slicers (airline, date, airport)
- Drill into details
- Export reports
